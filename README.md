The **Medical History Record System** is a Java-based application designed to manage patient medical history efficiently.  
It maintains patient details, medical records, and allergy information, and provides services to update and retrieve medical history.

This project demonstrates the use of **Core Java**, **OOP concepts**, and **collection framework**.
